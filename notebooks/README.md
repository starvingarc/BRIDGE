# Notebooks

This directory is reserved for formal notebook entrypoints and public-facing examples.

该目录用于存放正式 notebook 入口和公开示例。

Current `drafts/*.ipynb` research notebooks are not migrated into BRIDGE v1.

当前 `drafts/*.ipynb` 研究型 notebook 不会整体迁入 BRIDGE v1。

Migration rule:
- standardize
- document
- test
- then promote

迁移顺序：
- 标准化
- 文档化
- 可测试化
- 再提升为正式内容
