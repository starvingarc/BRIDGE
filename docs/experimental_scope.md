# Experimental Scope

The following content remains outside BRIDGE v1 formal scope:
- exploratory notebooks
- thesis-generation materials
- provisional plotting scripts
- unpublished ad hoc analyses

以下内容目前不属于 BRIDGE v1 正式范围：
- exploratory notebook
- 论文生成材料
- 临时绘图脚本
- 未发表的临时分析代码

These may be migrated later only after standardization, documentation, and test coverage.

这些内容只有在完成标准化、文档化与测试覆盖后，才会被迁移进正式仓库。
